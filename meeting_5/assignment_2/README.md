# Pengerjaan Tugas 2

**Soal:**

**Problem 2: Reverse an Array**

In this problem, you will write a function that takes a std:::vector<int> as input and returns a new
vector that contains the elements of the input vector in reverse order.

For example:
```bash
Input: [1, 2, 3, 4, 5]
Output:[5, 4, 3, 2, 1]
```
